The software which is used entirely through the HTC VIVE, which allows the user to see the object and modify the object as if it is a canvas. I found that this was the best for my 3D model as it was more dynamic. The one thing that frustrated me about this software was the accessibility,

Blender

This was the original software I started using, but I couldn't get it to work so I scrapped my model on this because I couldn't get it to look how I wanted it to look, but this was the software I best understood.
